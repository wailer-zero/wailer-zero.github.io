格式化输出
- 使用占位符


- format

占位符的种类
```
‘d’ 有符号十进制整数。

    'i'   有符号十进制整数。

    'o'   有符号八进制数。

    'u'   过时类型 -- 等价于 'd'。

    'x'   有符号十六进制数（小写）。

    'X'   有符号十六进制数（大写）。

    'e'   浮点指数格式（小写）。

    'E'   浮点指数格式（大写）。

    'f'   浮点十进制格式。

    'F'   浮点十进制格式。

    'g'   浮点格式。 如果指数小于 -4 或不小于精度则使用小写指数格式，否则使用十进制格式。

    'G'   浮点格式。 如果指数小于 -4 或不小于精度则使用大写指数格式，否则使用十进制格式。

    'c'   单个字符（接受整数或单个字符的字符串）。

    'r'   字符串（使用 repr() 转换任何 Python 对象）。

    's'   字符串（使用 str() 转换任何 Python 对象）。

    'a'   字符串（使用 ascii() 转换任何 Python 对象）。

    '%'   不转换参数，在结果中输出一个 '%' 字符。
    ```    
format的方法
    {1}{2}{3}调换顺序
    ```print('{0} {2} {1}'.format(movie, count, ticket))```实现的为movie, ticket, count.
    {a}{b}{c}
    ```print('{a}{b}{c}'.format(a='ni', c='a', b='hao'))```
